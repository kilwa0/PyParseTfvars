# PyParseTfvars

Parse tfvars files to use terraform variable values into python scripts

## Getting Started

```sh
git clone https://github.com/kilwa0/PyParseTfvars.git
```

## Installing

```sh
python3 -m pip install git+https://github.com/kilwa0/PyParseTfvars.git
```
